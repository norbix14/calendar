## Pasos
Clonar este repositorio

		git clone <repository>

Ejecutar **[APP](http://localhost:3000 "Modo de desarrollo")**

		npm start

Crear `build` de producción

		npm run build

## Elementos necesarios

Variable de entorno con la ruta al backend de la **API**

		REACT_APP_API_URL=""
